# FleetControl — Система управления автопарком

Веб-приложение для управления ремонтом и техническим обслуживанием корпоративного автопарка с ролевым доступом, темной темой и поддержкой двух языков.

## 🚀 Возможности

### Аутентификация и авторизация
- ✅ Полноценная система регистрации и входа
- ✅ Ролевой доступ (водитель, диспетчер, администратор)
- ✅ Защита маршрутов с автоматическим редиректом
- ✅ Сессии с сохранением в localStorage
- ✅ Управление профилем и смена пароля

### Темная тема
- ✅ Переключение между светлой, темной и системной темой
- ✅ Полная поддержка dark mode на всех 16 страницах
- ✅ Адаптивные цвета для всех UI компонентов
- ✅ Сохранение выбранной темы

### Локализация
- ✅ Русский и английский языки
- ✅ 436 переведенных ключей
- ✅ Переключение языка в реальном времени
- ✅ Сохранение выбранного языка

### Функциональность по ролям

#### 👤 Водитель
- Подача заявок на ремонт с фото и описанием
- История всех обращений с фильтрацией
- Уведомления о статусе заявок и плановом ТО
- Чат с диспетчерской службой
- Просмотр информации о закрепленном ТС

#### 📋 Диспетчер
- Очередь входящих заявок
- Назначение механиков на заявки
- Мониторинг водителей на линии
- Сводка по состоянию автопарка
- Управление статусами ремонта

#### 📊 Администратор
- Аналитическая панель с KPI
- Графики динамики заявок
- Распределение ТС по статусам
- Эффективность водителей
- Бюджет обслуживания

### Учетные записи

|------|-------|--------|--------|
| **Водитель** | `driver@fleet.ru` | `driver123` | `/dashboard` |
| **Диспетчер** | `dispatcher@fleet.ru` | `dispatcher123` | `/dispatcher` |

## ⚙️ Основные компоненты

### Контексты

#### AuthContext
- `login(email, password)` — Вход в систему
- `register(data)` — Регистрация нового пользователя
- `logout()` — Выход из системы
- `updateProfile(data)` — Обновление профиля
- `changePassword(current, new)` — Смена пароля
- `updateNotifPrefs(prefs)` — Настройки уведомлений

#### ThemeContext
- `theme` — Текущая тема (light/dark/system)
- `setTheme(theme)` — Установка темы
- `effectiveTheme` — Реальная тема с учетом системной

#### LocaleContext
- `locale` — Текущий язык (ru/en)
- `setLocale(locale)` — Установка языка
- `t(key)` — Функция перевода

### UI Компоненты

- `Button` — Кнопка с вариантами (primary, outline, ghost, danger)
- `Card` — Карточка контента
- `Input` — Текстовое поле
- `Select` — Выпадающий список
- `Textarea` — Многострочное поле
- `Badge` — Бейдж статуса
- `MetricCard` — Карточка метрики с иконкой
- `EmptyState` — Заглушка для пустых состояний
- `StatusDot` — Индикатор статуса

Все компоненты поддерживают dark mode.

## 🎨 Настройка темы

Темы настраиваются в `tailwind.config.ts`:

```typescript
colors: {
  brand: {
    50: '#eff6ff',
    // ... остальные оттенки
    600: '#2563eb',  // Основной цвет бренда
    // ...
  }
}
```

Dark mode классы в `globals.css`:

```css
.dark body {
  background: #0f172a;
  color: #e2e8f0;
}
```

## 🌍 Добавление переводов

Переводы находятся в `src/lib/i18n.ts`:

```typescript
export const translations: Record<Locale, Record<string, string>> = {
  ru: {
    'key.name': 'Русский текст',
  },
  en: {
    'key.name': 'English text',
  },
};
```

Использование:

```tsx
const { t } = useLocale();
<h1>{t('key.name')}</h1>
```
