# Carvix - AutoManager v1.0

## Информационная система управления ремонтом и техническим обслуживанием автопарка предприятия

![Carvix Logo](carvix_logo.png)

## 🚀 Быстрый старт

### Установка зависимостей

```bash
pip install PyQt6 matplotlib pandas reportlab
```

### Запуск приложения

```bash
python carvix_app.py
```

## 📋 Системные требования

- Python 3.8+
- PyQt6 6.0+
- SQLite3 (встроен в Python)
- Опционально: matplotlib, pandas, reportlab (для графиков и экспорта)

## 🗄️ База данных

База данных SQLite автоматически создается при первом запуске:
- `carvix_database.db` - файл базы данных

### Структура таблиц:
- `roles` - роли пользователей
- `users` - пользователи
- `vehicles` - транспортные средства
- `repair_types` - виды ремонта
- `maintenance_types` - виды ТО
- `defect_categories` - категории неисправностей
- `suppliers` - поставщики
- `parts` - запчасти
- `repair_requests` - заявки на ремонт
- `maintenance_requests` - заявки на ТО
- `request_parts` - использованные запчасти
- `request_works` - выполненные работы
- `audit_log` - журнал действий
- `notifications` - уведомления
